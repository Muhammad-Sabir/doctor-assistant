# Doctor Assistance - Frontend

## Run Application

1. **Navigate to the frontend project directory**:
    ```bash
    cd doctor-assistance-fe
    ```

2. **Install Dependencies:**
   Install all the required packages using:
   ```bash
   npm install
   ```

3. **Start the project**:
    ```bash
    npm run dev
    ```
